-------------------------

1) fork and clone


2) Check current branch:
$ git status
On branch master


$  git branch 
* main

3) create and move to a new branch:
$ git checkout -b "test/branch-something"
Switched to a new branch 'test/branch-something'

	
4) do stuff and commit:
$  git commit -m "New 'file_a'"


5) Push (first time)
$  git push --set-upstream origin test/branch-something 

5b) push (Second...)
$ git push -u origin test/branch-something 



6) Switch back to the main branch by command to switch between branches.

$ git branch 
  main
* test/branch-something


$ git checkout main
